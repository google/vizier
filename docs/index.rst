OSS Vizier Documentation
========================

Open Source (OSS) Vizier is a Python-based interface for blackbox optimization
and research, based on Google's original internal `Vizier`_, one of the first
hyperparameter tuning services designed to work at scale.

.. _`Vizier`: https://dl.acm.org/doi/10.1145/3097983.3098043

.. figure:: ./assets/oss_vizier_service.gif

   **OSS Vizier's distributed client-server system. Animation credit to Tom Small.**

Installation
------------

See https://github.com/google/vizier#installation for instructions on
installing OSS Vizier.
